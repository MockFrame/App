# MockFrame

Create professional device mockups for your App Store listings, design presentations, and portfolio showcases.

## About

MockFrame is a native iOS app that helps developers and designers create polished, consistent app screenshots without needing complex design software.

### Features

- Precision grid overlay for exact region selection
- Customizable status bar elements (battery, time, signal, carrier)
- Multiple export formats (JPEG, PNG)
- High-resolution export ready for App Store
- Works offline with no cloud processing
- Privacy-focused: images never leave your device

### Perfect For

- App Store screenshot creation
- Design portfolio presentations
- Client mockup deliverables
- UI/UX case studies
- Marketing materials

## Requirements

- iOS 14.0 or later
- iPhone or iPad
